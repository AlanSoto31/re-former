## Getting Started
- To get a local copy up and running follow these simple steps.

### Instructions
- Open the console
- Download or `git clone https://github.com/AlanSoto31/re-former.git`
- `cd re-former`
- run `bundle install`
- run `yarn install`
- run `rails server`
